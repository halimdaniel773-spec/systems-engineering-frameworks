# Systems Engineering Frameworks

A collection of functional software utilities, data parsers, and algorithmic frameworks built to automate risk management, analyze on-chain data distribution, and interface with LLMs.

---

## 🤖 Projects

### 1. Halim AI (`/halim-ai`)
An early-stage, voice-activated conversational AI assistant developed at age 14. 
- **Core Stack:** Python, `speech_recognition`, `pyttsx3`, OpenAI API (`text-davinci-003`).
- **Features:** High-speed offline local text-to-speech synthesis and custom identity intercept logic.

### 📊 2. RiskCalc CLI (`/risk-calc`)
A lightweight terminal-based risk optimization tool designed to compute exact capital sizing during fast live execution.
- **Features:** Automates position sizing to strictly maintain a 0.5%–1.0% risk cap per trade sequence.

### 🐋 3. DEX Whale Tracker Script (`/whale-tracker`)
A lightweight data parser built to stream and flag block transaction activity across high-volatility Solana network sectors.
- **Features:** Simulates incoming RPC block streams to capture major wallet movements instantly.
